# Security Policy

## Supported Versions

| Badges Version | XenForo Version | Supported                            |
| -------------- | --------------- |------------------------------------- |
| 1.0.7          | v2.1.x          |:x: End-of-Life, Please Update        |
| 2.x.x          | v2.2.x          |:white_check_mark: Actively Supported |

## Reporting a Vulnerability

Please report any security vulnerabilities to _**security -at- versobit.com**_; Please make sure "XF-Badges" is in your subject line.

When you disclose a security vulnerability:
1. We will assess the reported vulnerability.
2. We will fix the vulnerability (if it's related to our addon).
3. We will disclose the security vulnerability as an [issue](https://github.com/VersoBit/XF-Badges/issues) and reccomend urgent upgrade.

In the event the vulnerability is related to XenForo:
1. We will report our findings to [XenForo Ltd](https://xenforo.com/) in a detailed matter.
2. We will ensure your original findings/report is included in our disclosure to [XenForo Ltd](https://xenforo.com/).
